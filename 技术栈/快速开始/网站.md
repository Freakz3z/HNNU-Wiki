# 程序员必备网站导航

> 部分网站由 AI 推荐，请大家各取所需。

## 🌟 神中神

### [B站（bilibili）](https://www.bilibili.com/)

**不要不把B站当作一个学习网站！**

B站是中国最大的视频学习平台之一，拥有海量免费编程教程：

**为什么推荐：**
- 🎓 **完全免费** - 大量优质免费教程
- 📚 **内容全面** - 从入门到精通全覆盖
- 👨‍🏫 **名师云集** - 各大培训机构官方账号
- 💬 **弹幕互动** - 学习不孤单，有问题实时讨论

**推荐UP主：**
- **黑马程序员** - 系统、全面、专业
- **尚硅谷** - Java/前端/大数据
- **菜鸟教程** - 实战项目多
- **动力节点** - 深入技术原理
- **同济子豪兄** - Python/数据分析/AI

**使用建议：**
1. 搜索时加上"教程"、"实战"等关键词
2. 优先看播放量高、评论好的视频
3. 配合弹幕学习，遇到问题可提问
4. 1.5倍速观看，提高学习效率

## 1️⃣ 编程与算法

### 在线编程练习

#### [LeetCode](https://leetcode.com/) / [LeetCode中国](https://leetcode.cn/)
- **用途**：算法面试准备
- **特点**：
  - 中英文双语支持
  - 精选面试真题
  - 在线代码执行
  - 社区讨论活跃
- **适合**：准备技术面试的程序员

#### [牛客网](https://www.nowcoder.com/)
- **用途**：编程练习和求职
- **特点**：
  - 国内大厂题库
  - 模拟面试功能
  - 校招/社招信息
  - 编程竞赛
- **适合**：求职者和编程爱好者

#### [洛谷](https://www.luogu.com.cn/)
- **用途**：算法竞赛练习
- **特点**：
  - 中文界面友好
  - 题目难度分级
  - 精彩题解分享
  - 竞赛模拟
- **适合**：OI/ACM竞赛选手

### 算法学习

#### [OI Wiki](https://oi-wiki.org/)
- **用途**：算法竞赛知识库
- **特点**：
  - 中文详细教程
  - 图文并茂
  - 代码示例丰富
  - 持续更新
- **适合**：竞赛选手和算法学习者

#### [VisuAlgo](https://visualgo.net/zh)
- **用途**：算法可视化
- **特点**：
  - 动画演示算法过程
  - 支持中文
  - 交互式操作
- **适合**：初学者理解算法原理

### 互动学习

#### [Codecademy](https://www.codecademy.com/)
- **用途**：互动式编程课程
- **特点**：
  - 在线编写代码
  - 即时反馈
  - 多语言支持
- **适合**：编程初学者

#### [菜鸟教程](https://www.runoob.com/)
- **用途**：编程入门教程
- **特点**：
  - 中文简洁易懂
  - 在线编辑器
  - 涵盖多种技术
  - 完全免费
- **适合**：零基础入门

## 3️⃣ 开发与协作

### 代码托管

#### [GitHub](https://github.com/) ⭐⭐⭐⭐⭐
- **用途**：版本控制和代码托管
- **特点**：
  - 全球最大代码托管平台
  - 开源项目丰富
  - 协作开发便捷
  - GitHub Actions CI/CD
- **必用理由**：开发者必备，几乎所有开源项目都在这里

#### [Gitee](https://gitee.com/)
- **用途**：国内代码托管
- **特点**：
  - 访问速度快
  - 免费私有仓库
  - 中文界面
- **适合**：国内开发者

### 技术问答

#### [Stack Overflow](https://stackoverflow.com/)
- **用途**：编程问题解答
- **特点**：
  - 全球最大技术社区
  - 质量高
  - 解决问题快
- **提示**：搜索时英文效果更好

#### [SegmentFault](https://segmentfault.com/)
- **用途**：中文技术社区
- **特点**：
  - 中文问答
  - 技术文章
  - 招聘信息

## 💡 使用技巧

### 1. 充分利用搜索
- 遇到问题先搜索：Google > Stack Overflow > GitHub Issues
- 英文搜索结果通常更好

### 2. 善用文档
- 官方文档最权威
- 加入收藏夹，随时查阅

### 3. 参与社区
- GitHub上Star感兴趣的项目
- Stack Overflow上回答问题
- B站关注优质UP主

### 4. 保持学习
- 技术更新快，持续关注
- 订阅技术博客/Newsletter
- 参加线上/线下活动
